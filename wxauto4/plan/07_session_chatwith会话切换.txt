# 模块 07：会话切换 ChatWith / GetSession

目标：
实现稳定获取会话列表和切换聊天。

修改/新增文件：
1. wxauto4/ui/sessionbox.py
2. wxauto4/ui/main.py
3. wxauto4/wx.py
4. tests/test_session_chatwith_module07.py

GetSession 要求：
1. 基于 session_list.GetChildren()
2. 解析 mmui::ChatSessionCell
3. 返回会话名称、最后消息、时间、是否免打扰、未读数等可读字段
4. 解析失败返回 raw 信息

ChatWith(who, exact=False) 流程：
1. 先从当前可见 session_list 查找。
2. exact=True 时精确匹配。
3. exact=False 时允许包含匹配。
4. 找到后点击。
5. 找不到时使用搜索框：
   - EditControl(Name="搜索")
   - ClassName="mmui::XValidatorTextEdit"
6. 搜索框 Ctrl+A、粘贴 who。
7. 等待搜索结果。
8. 点击目标结果。
9. 切换后重新 init ChatBox。
10. 验证 chat_message_list 和 chat_input_field 存在。

测试：
python tests/test_session_chatwith_module07.py
python tests/test_session_chatwith_module07.py --name "文件传输助手"

默认只切换，不发送。
