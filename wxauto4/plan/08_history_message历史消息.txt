# 模块 08：历史消息读取

目标：
实现 GetHistoryMessage(n, callback=None, interval=0.5, speed=1, goback=True)。

修改/新增文件：
1. wxauto4/ui/chatbox.py
2. wxauto4/msgs/parser.py
3. tests/test_history_message_module08.py

实现思路：
1. 当前 msgbox 读取可见消息。
2. 向上滚动消息列表。
3. 每次滚动后读取新增消息。
4. 使用 message id 去重。
5. 达到 n 条后停止。
6. callback(message_list) 返回 True 时停止。
7. goback=True 时滚回原位置或底部。

风险：
1. UIAutomation 滚动可能不稳定。
2. 微信可能只加载部分历史。
3. 时间消息可能是特殊控件。
4. 不要疯狂滚动，避免卡死。

测试：
python tests/test_history_message_module08.py --n 50

验收：
能读取比当前可见更多的历史消息
不重复
不崩溃
