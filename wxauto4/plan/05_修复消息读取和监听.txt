- mmui::ChatTextItemView

要求：
1. self.msgbox 必须来自 chat_message_list。
2. 使用 self.msgbox.GetChildren() 读取消息项。
3. 每条消息至少保存 runtimeid、Name、ClassName、AutomationId、ControlType、Rect。
4. 原有消息解析逻辑能用就继续复用。
5. 如果某条消息无法解析，不要中断整个流程。
6. 无法解析的消息返回 raw message 信息。
7. 监听时使用 runtimeid 去重。
8. 切换聊天后重新初始化 msgbox。
9. AddListenChat / RemoveListenChat / StopListening 保持原有 API 不变。

建议实现：
def get_message_items(self):
    if not ctrl_exists(self.msgbox):
        self.init()
    if not ctrl_exists(self.msgbox):
        raise WxautoUINotFoundError("未找到消息列表 chat_message_list")
    return self.msgbox.GetChildren()

def get_message_key(item):
    try:
        return tuple(item.runtimeid)
    except Exception:
        return f"{item.Name}|{item.ClassName}|{rect}"

def parse_message_safe(item):
    try:
        return 原有解析逻辑(item)
    except Exception as e:
        return {
            "type": "raw",
            "name": safe_get(item, "Name"),
            "class_name": safe_get(item, "ClassName"),
            "automation_id": safe_get(item, "AutomationId"),
            "error": repr(e),
        }
