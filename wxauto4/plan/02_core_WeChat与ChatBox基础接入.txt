- ClassName: mmui::ChatMessagePage

message_view:
- ClassName: mmui::MessageView

chat_message_list:
- AutomationId: chat_message_list
- ClassName: mmui::RecyclerListView

- ClassName: mmui::InputView

- Name: 发送
- ClassName: mmui::XOutlineButton
- ControlType: ButtonControl

session_list:
- AutomationId: session_list
- ClassName: mmui::XTableView
- Name: 会话

# 模块 02：WeChat 与 ChatBox 基础接入

目标：
把 locator 层接入主窗口和聊天窗口初始化，使 WeChat() 能稳定获得当前聊天页核心控件。

修改文件：
1. wxauto4/ui/main.py
2. wxauto4/ui/chatbox.py
3. wxauto4/ui/sessionbox.py
4. tests/test_core_init_module02.py

main.py 要求：
1. 主窗口查找支持 ClassName=mmui::MainWindow。
2. 支持窗口名 微信 / Weixin / WeChat。
3. 不要只依赖窗口标题。
4. _setup_ui 中优先定位：
   - navigation: MainView.main_tabbar / mmui::MainTabBar
   - session_list: AutomationId=session_list
   - chat_page: chat_message_page

chatbox.py 要求：
1. ChatBox.init() 使用 locator。
2. 初始化：
   - self.chat_page
   - self.message_view
   - self.msgbox
   - self.input_view
   - self.editbox
   - self.sendbtn
   - self.tools
3. 保留旧逻辑 fallback。
4. sendbtn 找不到时不报错，因为输入框为空时按钮可能不显示。
5. msgbox 和 editbox 必须尽可能成功。

sessionbox.py 要求：
1. 优先使用 session_list。
2. 支持 ClassName=mmui::XTableView。
3. 支持会话 item：mmui::ChatSessionCell。

测试脚本 tests/test_core_init_module02.py：
1. 初始化 WeChat。
2. 打印主窗口状态。
3. 打印 ChatBox 控件状态。
4. 打印 SessionBox 控件状态。
5. 不发送消息。

测试命令：
python tests/probe_wechat_locator.py
python tests/test_core_init_module02.py

验收：
WeChat 初始化成功
chat_message_page OK
chat_message_list OK
chat_input_field OK
session_list OK
