核心要求：
1. 发送前确认 editbox 存在。
2. 点击 editbox。
3. 通过剪贴板粘贴文本，避免 SendKeys 中文输入不稳定。
4. 输入文本后重新定位 send_button。
5. 只点击 Name="发送" + ClassName="mmui::XOutlineButton" 或 Name="发送" 的按钮。
6. 不要点击：发送表情(Alt+E)、发送收藏、发送文件。
7. 发送后等待 0.2~0.5 秒。
8. 发送失败时抛出清晰异常或返回明确错误。

建议新增 ChatBox 方法：
def refresh_send_button(self):
    self.sendbtn = find_first(self.control, SELECTORS["send_button"], search_depth=30)
    if not ctrl_exists(self.sendbtn) and hasattr(self, "input_view") and ctrl_exists(self.input_view):
        self.sendbtn = find_first(self.input_view, SELECTORS["send_button"], search_depth=30)
    return self.sendbtn

SendMsg 建议流程：
1. if not ctrl_exists(self.editbox): self.init()
2. editbox.Click()
3. clear=True 时 Ctrl+A。
4. pyperclip.copy(msg)
5. editbox.SendKeys("{Ctrl}v")
6. time.sleep(0.2)
7. refresh_send_button()
8. 校验 sendbtn.Name == "发送"
9. sendbtn.Click()
10. time.sleep(0.3)
